# **App Name**: হিসাব খাতা

## Core Features:

- ব্যবহারকারী প্রমাণীকরণ (User Authentication): নিবন্ধন এবং লগইন কার্যকারিতা ব্যবহারকারীদের ব্যক্তিগত ডেটা সুরক্ষিত করতে।
- আয় যোগ করা (Income Entry): ব্যবহারকারীদের তাদের মাসিক বেতন এবং অন্যান্য আয় যুক্ত করার অনুমতি দিন।
- খরচ যোগ করা (Expense Tracking): ব্যবহারকারীদের তারিখ, বিভাগ এবং পরিমাণ সহ তাদের দৈনিক খরচ রেকর্ড করতে দিন।
- আয়-ব্যয়ের হিসাব দেখা (Overview Report): ব্যবহারকারীদের জন্য তাদের আয় এবং ব্যয়ের একটি সংক্ষিপ্ত দৃশ্য তৈরি করুন।
- কাস্টমাইজড বাজেট তৈরি (Custom Budgeting): ব্যবহারকারীরা তাদের প্রয়োজন অনুযায়ী বাজেট তৈরি করতে পারবে।
- ধারের হিসাব: ব্যবহারকারীরা তাদের বন্ধুদের থেকে ধার দেওয়া অথবা নেওয়া  ধারের হিসাব তৈরি করে রাখতে পারবে।

## Style Guidelines:

- Primary color: হালকা সবুজ (#90EE90), যা ব্যবহারকারীকে একটি আরামদায়ক অনুভূতি দেবে।
- Background color: প্রায় একই রকম একটি হালকা সবুজ (#E0FFDA) যা চোখের জন্য আরামদায়ক।
- Accent color: আর একটু গাঢ় সবুজ (#32CD32), যা প্রধান কাজ গুলো চিহ্নিত করতে সাহায্য করবে।
- Body এবং Headline এর জন্য: 'PT Sans', কারণ এটি দেখতে আধুনিক এবং সহজবোধ্য।
- অ্যাপটির লেআউটটি এমন হবে যাতে ব্যবহারকারীরা সহজে বুঝতে পারে এবং ব্যবহার করতে সুবিধা হয়।
- সবুজ এবং সাদা রঙের সহজ আইকন ব্যবহার করা হবে।
